## How to Use

1. **Open the App**: Simply open `index.html` in your web browser
2. **Create a Profile**: 
   - Go to the "Profiles" tab
   - Enter a profile name (e.g., "John", "Jane", "Mom", etc.)
   - Click "Create Profile"
   - The profile will be automatically selected
3. **Switch Profiles**: Use the profile dropdown in the header to switch between different users
4. **Start Tracking**: Once a profile is selected, the app automatically starts tracking from that day
5. **Daily Check-in**: Complete your daily check-in each day to maintain your streak
6. **Log Cravings**: Whenever you experience a craving, log it to track patterns
7. **Set Goals**: Create personal goals to work towards
8. **Stay Motivated**: Check the motivation tab for daily quotes and coping strategies
9. **Manage Profiles**: Go to the Profiles tab to create new profiles, switch between them, or delete profiles
