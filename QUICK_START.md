# ⚡ Quick Start Guide

## Get Started in 3 Steps

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Set Up MongoDB Connection

Create a `.env` file in the project root:

```env
MONGODB_URI=mongodb://localhost:27017/addiction-free
```

**For MongoDB Atlas (Cloud)**:
```env
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/addiction-free
```

### Step 3: Start the Server
```bash
npm start
```

Then open: **http://localhost:3000**

---

## What You Get

✅ **Permanent Data Storage** - All data saved to MongoDB  
✅ **Never Lose Data** - Data persists even after closing browser  
✅ **Multi-Device Access** - Access from any device/browser  
✅ **Offline Support** - Falls back to localStorage if server is down  
✅ **Multiple Profiles** - Track multiple users separately
